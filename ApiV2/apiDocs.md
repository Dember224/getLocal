## Request

### GET
### `/availableStates`

#### Response

##### 200
> Returns a list of states and years that getLocal has data for.

### GET 
### `/stateCsvDownload`

#### Request

- `state`
    The state that data is to be returned for.
- `year`
    The election year that data is to be returned for.

#### Response

##### 200

> Returns a CSV of all complete records that getLocal has for a state on the given year.

[CSV Link](https://hubs-api.herokuapp.com/stateCsvDownload/virginia/2023)

##### 400

> The given state or year are not available. Please check the /getAvailableStates endpoint for a list of available states and years.
